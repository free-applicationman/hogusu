ほぐす Ver.2.0.0
==================

GitHub Pagesで公開するための完成版です。

【収録内容】
- 15ステップの全身セルフストレッチ
- 全ステップを個別に実行できるメニュー
- 姿勢変更案内と開始前5秒カウント
- 左右切り替え音声
- キャット＆カウの3秒ガイド
- 応援メッセージ8種類
- 完了メッセージ5種類
- 1・7・30・50・100・180・365日の継続音声
- BGM：OFF／川のせせらぎ／森の音（初期値OFF）
- 連続日数・今月の実施日数・累計回数
- 月間カレンダー
- localStorageへの端末内保存
- PWA／ホーム画面追加
- オフラインキャッシュ

【GitHubへアップロードする方法】
1. このREADME.txtが入っているフォルダを開きます。
2. 中にある以下のファイルとフォルダを、GitHubのアップロード画面へまとめてドラッグします。
   - index.html
   - app.js
   - styles.css
   - manifest.webmanifest
   - sw.js
   - assets フォルダ
   - README.txt
3. ZIPファイルのままアップロードしないでください。
4. index.htmlはリポジトリ直下に置いてください。
5. Commit changesを押します。

【GitHub Pagesの設定】
1. リポジトリの Settings を開きます。
2. 左側の Pages を選びます。
3. Sourceを Deploy from a branch にします。
4. Branchを main、フォルダを / (root) にします。
5. Saveを押します。

公開URL:
https://free-applicationman.github.io/hogusu/

【iPhoneのホーム画面へ追加】
1. Safariで公開URLを開きます。
2. 共有ボタンを押します。
3. 「ホーム画面に追加」を選びます。

【記録について】
実施記録はブラウザのlocalStorageに保存されます。
Safariの履歴・Webサイトデータを削除した場合や、別の端末・別のブラウザでは記録が引き継がれません。

【更新時の注意】
ファイル名やassets内のフォルダ階層は変更しないでください。
